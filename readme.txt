サイト監視サービス（MonitoringPlus）と連携し、WordPress管理画面で監視結果と監視アラートを確認

== Description ==

[サイト監視サービス（MonitoringPlus）](https://www.monitoring-plus.jp/ "MonitoringPlus")と連携し、WordPress管理画面で監視結果と監視アラートを見ることができます。

プラグインの詳細については、以下のページに記載しております。  
[プラグイン説明ページ](https://www.monitoring-plus.jp/plugin/wordpress/ "Documentation in Japanese")

MonitoringPlusプラグイン は MITライセンスの下で利用可能なWordPressプラグインです。  
WordPress ではプラグインやテーマの組み合わせにより機能の競合が起こる場合があります。  
問題が起きた場合はプラグインを無効にするなどして様子をみてください。

== Installation ==

1. WordPressプラグインディレクトリ よりインストールまたは、プラグインをダウンロードして `/wp-content/plugins/` ディレクトリへ直接インストール
2. プラグインページで、MonitoringPlusを有効化
3. メニュー内の "監視画面" を開き、MonitoringPlusとの連携承認を行う（事前にMonitoringPlusへの会員登録が必要です。会員登録は[こちら](https://www.monitoring-plus.jp/auth/create "MonitoringPlus会員登録")から）
